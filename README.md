A sleek and modern stopwatch application built with HTML, CSS, and JavaScript. Features a clean dark theme with precise time tracking down to milliseconds.

## Live Demo

[View Live Demo](https://spaciousejar.github.io/stopwatch/)

## Features

- **Precise Timing** — Tracks time with millisecond accuracy
- **Clean Interface** — Modern dark theme with circular stopwatch design
- **Responsive Design** — Works seamlessly across different screen sizes
- **Simple Controls** — Easy-to-use Start, Stop, and Reset buttons
- **Visual Appeal** — Elegant design with smooth animations and styling

## How to Use

1. Click **Start** to begin timing
2. Click **Stop** to pause
3. Click **Reset** to clear the timer

## Tech Stack

- **HTML5** — Structure
- **CSS3** — Styling and animations
- **JavaScript** — Timer logic and interactivity
